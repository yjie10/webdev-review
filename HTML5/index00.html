All HTML documents must start with a <!DOCTYPE>* declaration.
The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.
<!DOCTYPE html> is the declaration for HTML5.
- W3 Schools: HTML <!DOCTYPE> Delcaration

Self-closing tags include <br> (line break) and <hr> (horizontal line).
The syntax <br /> is from XHTML, a stricter and more XML-based version of HTML.

Image tag is also a self-closing tag.
