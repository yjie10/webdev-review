JavaScript Types

  1. Number
  2. String
  3. Boolean
  4. Undefined
  5. Null
  6. Symbol (new in ES6)
  7. Object
Number: A number.
NaN stands for Not a Number and it is a part of the Number type.
String: can use single ('') or double ("") quote.
In case have to use quotes in text, to avoid syntax errors, can use '\' (back slash) before the quote.

e.g. 'This isn't an apple. ' -> 'This isn\'t an apple. '

The back slash has a special meaning that it ignores the character that comes after it in some cases.

Number + String = String
Number -*/ String = Number
Boolean: has a value of true or false.
Undefined: variable unassigned a value yet.
